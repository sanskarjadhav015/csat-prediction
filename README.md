# 📊 CSAT Prediction Project

## 📌 Overview
This project predicts Customer Satisfaction (CSAT) using Machine Learning.

## 🚀 Features
- Data preprocessing
- Model training
- Prediction system

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

## 📂 Project Structure
- app.py
- model.pkl
- requirements.txt

## ▶️ How to Run
```bash
pip install -r requirements.txt
python app.py
```

## 📈 Output
Predicts customer satisfaction score based on input data.
